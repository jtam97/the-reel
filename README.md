# THE REEL

A retro cinematic movie dashboard built with Next.js. Explore upcoming releases on a horizontal film strip timeline, browse a calendar of what's hitting theaters, and discover what happened on any date in cinema history.

## Features

- **Now Showing** — Timeline and calendar views of current and upcoming releases, including highly anticipated future films
- **The Vault** — Pick any month and day to see which movies released on that date throughout history, with superlative awards (Box Office King, Critics' Darling, Best Investment, and more)
- **Retro Theme** — Film grain, flickering marquee text, sprocket holes, ticket stub buttons, and gold glow effects

## Getting Started

1. Get a free API key from [TMDB](https://www.themoviedb.org/settings/api)
2. Create `.env.local` and add your key:
   ```
   NEXT_PUBLIC_TMDB_API_KEY=your_key_here
   ```
3. Install and run:
   ```bash
   npm install
   npm run dev
   ```
